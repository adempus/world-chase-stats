# world-chase-stats

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

docker run -d world_chase_stats_container -p 8080:8080 world-chase-stats
